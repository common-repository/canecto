The easy way to add your Canecto code to your WP site.

== Description ==

This plugin makes it very fast, simple, and easy to your Canecto script code to your WordPress site. It's great for non-geeky people who would like to simply copy and paste the code Canecto provides into WordPress (see the screenshots). It's also great for web professionals who are looking for a fast way to add the Canecto scripts to client websites.

= How to use canecto Connecticator =

*You must have a Canecto account to use this plugin*

1. Get your Canecto code from your Canecto account.

2. Now go to your WordPress dashboard.

3. Find the plugin in the WordPress menu under Settings > Canecto Connecticator

4. Just paste your code in the box and click the Save button.

5. Now open up a page of your website, to trigger a page view in Canecto.

6. Finally test your script is working in your Canecto account.

== Installation ==

You can find the plugin in the WordPress menu under Settings > Canecto Connecticator

= Automated =

1. Go to the Plugins menu in WordPress
2. Search for `Canecto Connecticator`
3. Install and activate the plugin

= Manual =

1. Upload `canecto` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress
3. Configure plugin (go to the Settings/Canecto Connecticator)
